# Stock Price Notifier

## Steps to run the code:

- Clone the repo:
```bash
  git clone https://github.com/rravii/stock_price_notifier.git
```
- Install the requirements file
```bash
  pip install -r requirements.txt
```
- Run the code
```bash
  python run.py
```
